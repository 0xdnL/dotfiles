# dotfiles

> repo for tracking my dotfiles from $HOME

## prerequesites

- git
- chezmoi

## usage

```sh
chezmoi init

chezmoi add FILE

chezmoi cd

git add .
git commit -v
git push

exit

chezmoi diff

# add other steps later..
```
