# dotfiles

> repo for tracking my dotfiles from $HOME

## prerequesites

- git
- homebrew
- ..

## usage

```sh
cd $HOME

git init

git remote add origin git@github.com:dnltinney/dotfiles.git

git fetch

git checkout $(hostname -s)
```
